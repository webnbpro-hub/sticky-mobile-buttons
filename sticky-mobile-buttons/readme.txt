== Description ==

Sticky Mobile Buttons is a WordPress plugin that adds fully customizable sticky buttons to the bottom of your website on mobile devices. Perfect for improving mobile user experience with quick access to contact options and shopping cart.

= Features =
- Up to 7 customizable buttons
- Multiple button types: Phone, Telegram, WhatsApp, Cart, and custom links
- Visual icon selector with Font Awesome icons
- Custom icon upload support
- Color customization for each icon
- Show/hide text labels
- Responsive design - only shows on mobile devices
- Smooth animation effects

== Installation ==

1. Upload the `sticky-mobile-buttons` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings -> Sticky Buttons to configure your mobile buttons

== Frequently Asked Questions ==

= How many buttons can I add? =
You can add up to 7 buttons.

= Can I use custom icons? =
Yes, you can choose from Font Awesome icons or upload your own custom images.

= Does this work with page builders? =
Yes, the plugin works with any WordPress theme and page builder.
